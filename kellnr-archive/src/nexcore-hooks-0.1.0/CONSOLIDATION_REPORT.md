### 6. Safe Regex Builder → `SafeRegex`

**Source**: Multiple binaries with identical safety patterns

**Before** (repeated in 15+ hooks):
```rust
fn make_re(pattern: &str) -> Regex {
    Regex::new(pattern).unwrap_or_else(|_| {
        Regex::new(".").unwrap_or_else(|_| {
            std::process::exit(1);
        })
    })
}
```

**After**: `src/patterns/mod.rs`
```rust
pub fn safe_regex(pattern: &str) -> Regex {
    Regex::new(pattern).unwrap_or_else(|e| {
        eprintln!("Invalid regex pattern '{}': {}", pattern, e);
        // Fallback to never-match pattern
        Regex::new(r"^\b$").expect("fallback regex is valid")
    })
}
```

**Impact**: DRY principle, consistent error handling.
